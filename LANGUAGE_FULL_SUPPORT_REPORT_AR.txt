تقرير التعديل: دعم اللغات الكامل

النسخة المعدلة مبنية على: main-app-v6-7-user-profile-messages-auth.zip

التعديلات:
- إضافة طبقة ترجمة عامة للتطبيق داخل src/lib/i18nDom.js.
- دعم العربية / الفرنسية / الإنجليزية لكل النصوص الظاهرة في الواجهة.
- العربية تبقى اللغة الافتراضية.
- اختيار اللغة يبقى داخل صفحة حساب المستخدم فقط.
- ترجمة الصفحة الرئيسية، الأقسام، صفحة السيارة، صفحة الوكالة، التسجيل، الدخول، أضف وكالتك، الرسائل، الإشعارات، رحلاتي، المفضلة، الفلاتر، الموقع، المقدمة، وBottom Sheets.
- ترجمة placeholders و aria-label و alt/title عندما تحتوي على نص عربي.
- تغيير اتجاه الصفحة تلقائيًا: RTL للعربية و LTR للفرنسية/الإنجليزية.
- ترجمة تواريخ صفحة السيارة حسب اللغة المختارة.

فحص:
- تم فحص ملف الترجمة الجديد عبر node --check بنجاح.
- تم فحص الاستيرادات الداخلية وعدم وجود import مفقود.
- لم يتم تشغيل npm run build داخل هذه البيئة بسبب تعطل الوصول إلى npm registry، لكن بنية Vercel بقيت كما هي في النسخة التي كانت تعمل عندك.

إعدادات Vercel:
Framework Preset: Vite
Install Command: npm install --registry=https://registry.npmjs.org/
Build Command: npm run build
